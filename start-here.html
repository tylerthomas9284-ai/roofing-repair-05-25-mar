🔍 If You Don't See Styling

Common Issue: CSS Not Loading

Symptoms: Everything is black text on white background, no colors or styling

Solution 1: Check File Location

  1. Open your file explorer
  2. Navigate to the public folder
  3. Verify styles/main.css exists
  4. Verify js/main.js exists

Solution 2: Use Browser DevTools

  1. Right-click on the page → "Inspect"
  2. Go to Console tab
  3. Look for red errors
  4. If you see Failed to load resource: styles/main.css → CSS path is wrong!

Solution 3: Check Network Tab

  1. Open DevTools (F12)
  2. Go to Network tab
  3. Refresh the page (F5)
  4. Find main.css in the list
  5. Status should be 200 (Success)
  6. If it's 404 (Not Found) → check file structure!

Solution 4: Hard Refresh

Clear browser cache and reload:

  • Windows: Ctrl + Shift + R or Ctrl + F5
  • Mac: Cmd + Shift + R
  • Linux: Ctrl + Shift + R

Solution 5: Use Local Server (Recommended)

Instead of opening files directly, run a local server:

Using Python (Easiest):

cd public
python -m http.server 8000

Then open: http://localhost:8000

Using Node.js:

cd public
npx http-server -p 8000

Then open: http://localhost:8000
